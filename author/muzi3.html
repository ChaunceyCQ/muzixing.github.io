1：启动pox.py。

Pox.py 里面除了一堆的注释以外，真正有用的只有几句会运行的：

from pox.boot import boot
    if __name__ == '__main__':
        boot()

boot()函数在pox.boot里，有什么内容呢？

1：把pox和ext两个文件夹的路径加入系统path中。
2：由_do_ import添加各个模块
